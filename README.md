# 💱 Currency Converter

A simple and responsive currency converter web app that gives real-time exchange rates.

## 🔗 Live Demo
[Click here to view] (https://Aanya-Jain07.github.io/Currency_Converter)

## ✨ Features
- Real-time exchange rates using ExchangeRate API
- Supports 150+ currencies with country flags
- Swap button to quickly switch between currencies
- Responsive design — works on all devices
- Auto-fetches rate on page load

## 🛠️ Built With
- HTML
- CSS
- JavaScript
- [ExchangeRate API](https://www.exchangerate-api.com)
- [Flags API](https://flagsapi.com)
- [Font Awesome](https://fontawesome.com)


## 🚀 How to Use
1. Enter the amount
2. Select **From** currency
3. Select **To** currency
4. Click **Get Exchange Rate**
